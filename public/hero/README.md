# Hero Slideshow Images

Place your custom slideshow images here.

## File names (exact):

| File           | Slide label     |
|----------------|-----------------|
| slide-1.jpg    | Slide 1         |
| slide-2.jpg    | Slide 2         |
| slide-3.jpg    | Slide 3         |
| slide-4.jpg    | Slide 4         |
| slide-5.jpg    | Slide 5         |

## Tips
- Recommended size: 1920 × 1080 px (16:9)
- Format: JPG or PNG (rename to .jpg either way)
- The hero overlays a dark gradient on top, so dark/moody images work best
- If a file is missing, the slide falls back to a default Unsplash image automatically

## Editing slide labels
Open `components/landing/hero.tsx` and edit the `label` field in the `SLIDES` array.
